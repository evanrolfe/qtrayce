# QTrayce

## Setup
```bash
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run
```
make install
make run
```

## Test
```
make test
```
Or run specific test(s):
```
make test -- -k describe_containers_dialog
```
